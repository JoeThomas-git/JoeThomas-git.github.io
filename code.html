    How to install the 'joethomasRfunctions' package

    Tutorials based on functions written for my R package (joethomasRfunctions).

    Before you start, make sure to install and require the "joethomasRfunctions" package using devtools.

    Install instructions:

    1. Install and require the devtools package.

    install.packages('devtools')


    library(devtools)



    2. Download and install the 'joethomasRfunctions' package.

    devtools::install_github("joethomas-git/joethomasRfunctions")



    3. Require the 'joethomasRfunctions' package.

    library(joethomasRfunctions)
